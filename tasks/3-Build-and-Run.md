# 3. Build and Run

## Objective
Compile and execute the C# console calculator application

## Steps
1. Open terminal/command prompt in project directory
2. Build the application:
   ```bash
   dotnet build
   ```
3. Run the application:
   ```bash
   dotnet run
   ```
4. Follow on-screen prompts to test calculator functionality

## Tips
- Use `dotnet --version` to verify SDK installation
- Ensure no other instance of the app is running
- Check for build errors in the terminal output
